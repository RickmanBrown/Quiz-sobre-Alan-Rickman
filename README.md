# Alan Rickman Quiz Java

Um quiz em **Java** com **10 perguntas** sobre Alan Rickman, criado para praticar lógica de programação, entrada de dados com `Scanner` e estruturas condicionais.

## Sobre o projeto

Este projeto é um quiz de terminal feito em Java.  
Ao iniciar o programa, o usuário escolhe se deseja jogar ou não.  
Se escolher jogar, responde a **10 perguntas** sobre Alan Rickman e, ao final, recebe sua pontuação.

## Funcionalidades

- Menu inicial para começar ou sair do quiz
- 10 perguntas sobre Alan Rickman
- Contagem de acertos
- Exibição da nota final no terminal
- Encerramento do programa ao final

## Tecnologias utilizadas

- Java
- Eclipse

## Estrutura do projeto

O projeto possui uma classe principal:

- `App.java` — responsável por executar o quiz

## Como executar

1. Abra o projeto em alguma **IDE**
2. Localize a classe `App.java`
3. Execute o arquivo como **Java Application**
4. Responda às perguntas digitando o número da alternativa desejada
